# 🏥 AI HealthyCare Chatbot System (Flask + MySQL)

An **AI-powered healthcare chatbot application** developed to assist users with preliminary symptom checking, automated doctor recommendations, and seamless medical appointment bookings.
Built using **Python, Flask, and Natural Language Processing (NLP)**.

---

## 🚀 Key Features

### 🧠 NLP-Powered Symptom Checker

Uses Natural Language Processing to analyze user symptoms and predict potential medical conditions.

### 👨‍⚕️ Doctor Recommendation System

Smart algorithm that suggests the most suitable healthcare specialists based on identified symptoms.

### 📚 Automated Knowledge Base

A secure and structured medical repository used by the chatbot for accurate responses.

### 📅 Secure Appointment Booking

Allows users to book medical appointments with integrated **PDF generation** for confirmations and receipts.
